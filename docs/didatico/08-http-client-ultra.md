# 08 - httpClient Ultra Detalhado (PT-BR)

## Objetivo
Padronizar chamadas HTTP do frontend para backend.

## Estrutura
- Base URL da API.
- Config de timeout (opcional).
- Interceptors (opcional).

## Boas práticas
1. Usar `VITE_API_BASE_URL`.
2. Não hardcodear IP no código.
3. Trocar URL por ambiente:
   - desenvolvimento local
   - mobile
   - produção
